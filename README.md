# JavaScript Learning Journey 

## Day 3 - Operators in JavaScript

Today I learned about JavaScript operators and how they are used to perform different types of operations on values and variables.

### Topics Covered Today 

#### Arithmetic Operators
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Exponentiation (**)

#### Assignment Operators
- =
- +=
- -=
- *=
- /=
- %=

#### Comparison Operators
- ==
- ===
- !=
- !==
- >
- <
- >=
- <=

#### Logical Operators
- && (AND)
- || (OR)
- ! (NOT)

#### Increment & Decrement
- ++
- --

### Practice
- Solved basic operator examples.
- Compared values using comparison operators.
- Used logical operators in conditional expressions.

 Key Takeaways
- `==` checks value only.
- `===` checks both value and data type.
- Logical operators help combine multiple conditions.
- Assignment operators make code shorter and cleaner.
